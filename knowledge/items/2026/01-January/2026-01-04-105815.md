# React Hooks 深入理解

## 概述
React Hooks 是 React 16.8 引入的新特性,让函数组件拥有状态和其他 React 特性。

## 常用 Hooks

### useState
状态管理 Hook。

```javascript
const [count, setCount] = useState(0);
```

### useEffect
副作用处理 Hook。

```javascript
useEffect(() => {
  document.title = `Count: ${count}`;
}, [count]);
```

### useContext
上下文共享 Hook。

### useReducer
复杂状态管理 Hook。

## Hooks 规则
1. 只在函数顶层调用
2. 只在 React 函数中调用
3. 使用自定义 Hook 复用逻辑

## 优势
- 代码更简洁
- 逻辑复用更容易
- 副离更清晰
- 不需要 class

## 学习要点
- 理解 Hooks 原理
- 掌握常用 Hooks
- 遵循 Hooks 规则
- 自定义 Hooks
