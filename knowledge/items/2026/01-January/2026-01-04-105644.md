# React 基础概念

## 概述
React 是一个用于构建用户界面的 JavaScript 库,由 Facebook 开发。

## 核心概念

### 组件
React 应用由组件组成,组件是可复用的 UI 片段。

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```

### JSX
JSX 是 JavaScript 的语法扩展,允许在 JavaScript 中编写类似 HTML 的代码。

### Props
Props 是组件的输入,用于传递数据。

### State
State 是组件的内部状态,可以随时间变化。

## 虚拟 DOM
React 使用虚拟 DOM 来高效更新界面。

## 学习要点
- 组件化思维
- 单向数据流
- 声明式编程
