# JavaScript 异步编程

## 概述
JavaScript 是单线程语言,异步编程是处理耗时操作的关键。

## 核心概念

### 回调函数
最基础的异步处理方式。

### Promise
Promise 是异步操作的容器,代表未来可能完成的事件。

```javascript
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('操作完成');
  }, 1000);
});
```

### async/await
基于 Promise 的语法糖,让异步代码看起来像同步代码。

```javascript
async function fetchData() {
  const result = await promise;
  console.log(result);
}
```

## 事件循环
JavaScript 的事件循环机制是异步编程的基础。

## 学习要点
- 理解事件循环
- Promise 链式调用
- 错误处理
- 避免回调地狱
