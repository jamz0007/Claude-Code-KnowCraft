# 函数组件最佳实践

## 概述
函数组件是现代 React 开发的首选方式。

## 最佳实践

### 1. 保持简洁
```jsx
// ✅ 好的实践
function Button({ onClick, children }) {
  return <button onClick={onClick}>{children}</button>;
}

// ❌ 避免
function Button({ onClick, children, isLoading, disabled, variant, size, ... }) {
  // 逻辑过于复杂
}
```

### 2. 使用解构
```jsx
function UserProfile({ name, email, avatar }) {
  return (
    <div>
      <img src={avatar} alt={name} />
      <h3>{name}</h3>
      <p>{email}</p>
    </div>
  );
}
```

### 3. Props 默认值
```jsx
function Button({ variant = 'primary', size = 'medium' }) {
  // ...
}
```

### 4. 性能优化
```jsx
import { memo } from 'react';

const ExpensiveComponent = memo(function ExpensiveComponent({ data }) {
  // 只在 data 变化时重新渲染
});
```

## 使用场景
- 简单的 UI 组件
- 展示型组件
- 容器组件
- 自定义 Hooks

## 注意事项
- 避免过度组件化
- 合理使用 Hooks
- 注意性能优化
