# React Hooks 深入理解

## 概述
React Hooks 是 React 16.8 引入的新特性,让函数组件拥有状态和其他 React 特性。

## 常用 Hooks

### useState
状态管理 Hook。

```javascript
const [count, setCount] = useState(0);
```

使用 useState 可以在函数组件中添加状态。状态是组件的记忆。

### useEffect
副作用处理 Hook。

```javascript
useEffect(() => {
  document.title = `Count: ${count}`;
  return () => {
    // 清理函数
  };
}, [count]);
```

useEffect 用于处理副作用,如数据获取、订阅、DOM操作等。

### useContext
上下文共享 Hook,用于跨组件传递数据。

### useReducer
复杂状态管理 Hook,类似 Redux。

## Hooks 规则
1. 只在函数顶层调用
2. 只在 React 函数中调用
3. 使用自定义 Hook 复用逻辑

## 优势
- 代码更简洁
- 逻辑复用更容易
- 副离更清晰
- 不需要 class

## 学习要点
- 理解 Hooks 原理
- 掌握常用 Hooks
- 遵循 Hooks 规则
- 自定义 Hooks

## 性能优化
- 使用 useMemo 缓存计算结果
- 使用 useCallback 缓存函数
- 避免不必要的重渲染
